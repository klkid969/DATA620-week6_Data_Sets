# DATA620 Week 6 – Davis Southern Women Network Analysis

This assignment analyzes the Davis Southern Women dataset using NetworkX. The dataset represents attendance relationships between 18 women and 14 social events.

The original dataset forms a bipartite network where women are connected to the events they attended.

Using NetworkX, the bipartite network is projected into two separate networks:

• A women's network showing which women attended events together  
• An event network showing which events share the same attendees

Degree centrality is calculated to identify the most connected individuals within the social network.

The analysis and code are contained in the Jupyter Notebook included in this repository.
